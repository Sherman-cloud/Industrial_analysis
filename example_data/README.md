# 新能源汽车行业分析系统 - 示例数据

本目录包含用于演示系统的示例数据文件。

## 文件说明

### 1. 宏观经济数据 (macro_economic_data.csv)
包含中国宏观经济指标，如GDP、CPI、PMI等。

### 2. 行业销售数据 (industry_sales_data.csv)
包含新能源汽车行业销售数据，如月度销量、市场份额等。

### 3. 企业财务数据 (company_financial_data.csv)
包含主要新能源汽车企业的财务数据，如营收、利润、研发投入等。

### 4. 政策文件 (policy_documents.csv)
包含新能源汽车相关政策文件信息。

### 5. 市场调研数据 (market_research_data.csv)
包含消费者调研数据，如购买意愿、品牌偏好等。

### 6. 技术指标数据 (technology_indicators.csv)
包含新能源汽车技术指标，如电池能量密度、续航里程等。

## 使用方法

1. 将这些示例数据复制到 `data` 目录
2. 运行 `python main.py` 进行完整分析
3. 或运行 `python run_example.py` 查看示例分析

## 注意事项

- 这些数据仅用于演示目的，不代表真实市场情况
- 在实际使用中，请替换为真实数据
- 确保数据格式与系统要求一致
